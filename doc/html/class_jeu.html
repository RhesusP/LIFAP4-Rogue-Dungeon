void Jeu::initTerrain ( int  terNum)

Fonction initialisant un terrain depuis un fichier situé dans le sous-dossier terrain du dossier assets. Le nom du fichier est de la forme ter(terNum). Cette fonction lit dans le fichier la disposition des ennemis, de personnage et des obstacle. Elle initialise tous les attributs de terrain selon le niveau et génère aléatoirement le loot des ennemis.

Parameters
terNum(int) l'entier qui sera concaténé à "ter" pour ouvrir le fichier terrain correspondant
