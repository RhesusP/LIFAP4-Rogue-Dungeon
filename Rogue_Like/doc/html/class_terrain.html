◆ getTab()

char * Terrain::getTab ( )

Accesseur du tableau composant le Terrain. Le tableau sert à gérer le niveau et à être lu par les différentes classes. Chaque caractère correspond à un élément :

  • 'o' : obstacle
  • 'P' : personnage (héros)
  • 'E' : ennemi
  • 'v' : sol
  • 's' : petite potion
  • 'm' : moyenne potion
  • 'l' : grosse potion
  • 'w' : arme
  • 'a' : armure
  • 'S' : sortie
  • 'r' : le sol rouge laissé par le boss là où il passe
    Returns
    Tableau de caractères.
